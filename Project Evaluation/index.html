เครื่องมือสร้าง Prompt วิเคราะห์ผลประเมิน

กรอกข้อมูลเบื้องต้นและกด "สร้าง Prompt" เพื่อเตรียมข้อมูล จากนั้นคัดลอกและนำไปใช้กับ AI อย่าง ChatGPT, Claude หรือ Gemini ได้เลย

ตั้งค่าข้อมูลโครงการ

ขั้นตอนการใช้งาน:
1. กรอกข้อมูลให้ครบ
2. กด "สร้าง Prompt สำหรับ AI"
3. กด "คัดลอก Prompt" และนำไปวางใน AI พร้อมแนบไฟล์ Excel
