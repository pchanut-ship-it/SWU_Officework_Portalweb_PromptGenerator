SWU Prompt Generator

เครื่องมืออำนวยความสะดวกในการสร้าง Prompt สำหรับให้ AI ช่วยร่าง หลักการและเหตุผล ที่เชื่อมโยงกับทิศทางและพันธกิจของมหาวิทยาลัยศรีนครินทรวิโรฒ

ข้อมูลโครงการ

กรอกข้อมูลเบื้องต้นเพื่อใช้เป็นฐานข้อมูลในการสร้าง Prompt

คำแนะนำ: กรุณาระบุชื่อโครงการ หรือ วัตถุประสงค์ เพื่อให้ระบบเขียน Prompt ได้อย่างสมบูรณ์ที่สุด

ผลลัพธ์โครงสร้างคำสั่ง

คัดลอกข้อความไปวางลงใน ChatGPT, Claude หรือ Gemini

พร้อมช่วยเหลือคุณสร้างสรรค์โครงการระดับมหาวิทยาลัย

กรอกรายละเอียดโครงการและกดปุ่มสร้าง Prompt ได้ทันที

แนะนำเพิ่มเติมสำหรับผู้ใช้

เมื่อนำ Prompt นี้ไปส่งต่อให้ AI คุณจะได้รับแบบร่าง "หลักการและเหตุผล" ที่มีความน่าเชื่อถือทางวิชาการและเป็นทางการสูง หากต้องการเพิ่มจุดเน้นส่วนตัว เช่น "เน้นการพัฒนาทักษะในศตวรรษที่ 21" หรือ "มุ่งเน้นการรับใช้สังคมแบบยั่งยืน" สามารถพิมพ์ปรับปรุงต่อในกล่อง AI ได้อย่างอิสระครับ
